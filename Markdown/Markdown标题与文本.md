# Markdown标题与文本

## 1. 标题语法

### 六级标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

说明：使用#号表示标题级别，最多6级，#后必须加空格

## 2. 文本格式化

### 加粗

```markdown
**加粗文本** 或 __加粗文本__
```

说明：双星号或双下划线包裹文本实现加粗效果

### 斜体

```markdown
*斜体文本* 或 _斜体文本_
```

说明：单星号或单下划线包裹文本实现斜体效果

### 加粗斜体

```markdown
***加粗斜体*** 或 ___加粗斜体___
```

说明：三个星号或下划线同时实现加粗和斜体

### 删除线

```markdown
~~删除的文本~~
```

说明：双波浪线包裹文本添加删除线效果
