# Markdown表格与其他

## 1. 表格

### 基本表格

```markdown
| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25 | 北京 |
| 李四 | 30 | 上海 |
```

说明：竖线分隔列，第二行的横线分隔表头和内容

### 对齐方式

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 内容 | 内容 | 内容 |
```

说明：冒号在左侧左对齐、两侧居中、右侧右对齐

### 表格中包含其他元素

```markdown
| 链接 | 代码 | 加粗 |
|------|------|------|
| [百度](https://www.baidu.com) | `code` | **bold** |
```

说明：表格单元格内可使用链接、代码、强调等Markdown语法

## 2. 分割线

### 基本语法

```markdown
---

***

___
```

说明：三个或以上减号、星号、下划线创建水平分割线

## 3. 转义字符

### 特殊字符转义

```markdown
\*不是斜体\*
\#不是标题\#
\[不是链接\]
```

说明：反斜杠转义特殊字符，使其作为普通文本显示

### 常用转义字符

```markdown
\` 反引号
\* 星号
\_ 下划线
\{\} 花括号
\[\] 方括号
\(\) 圆括号
\# 井号
\+ 加号
\- 减号
\. 点号
\! 感叹号
```

说明：所有Markdown特殊字符都可通过反斜杠转义

## 4. HTML标签混合

### 直接使用HTML

```markdown
<p style="color:red">红色文字</p>
```

说明：Markdown兼容HTML，可直接使用HTML标签实现复杂样式

### 换行

```markdown
<br>
```

说明：使用HTML的br标签强制换行

### 折叠内容

```markdown
<details>
<summary>点击展开</summary>
隐藏内容
</details>
```

说明：使用HTML的details和summary创建可折叠内容块

## 5. 自动链接

### URL自动识别

```markdown
https://www.example.com
```

说明：直接输入URL会自动转换为链接

### 邮箱自动识别

```markdown
email@example.com
```

说明：直接输入邮箱地址会自动转换为mailto链接

### 尖括号包裹

```markdown
<https://www.example.com>
```

说明：使用尖括号包裹URL或邮箱也可创建自动链接
