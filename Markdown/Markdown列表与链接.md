# Markdown列表与链接

## 1. 无序列表

### 基本语法

- 项目1
- 项目2
- 项目3

说明：使用减号、星号或加号创建无序列表，符号后需空格

### 其他符号

* 项目1
* 项目2

+ 项目1
+ 项目2

说明：-、*、+三种符号效果相同，可混用但不推荐

### 嵌套列表

- 项目1
  - 子项目1
  - 子项目2
- 项目2

说明：使用两个空格或Tab缩进创建子列表

## 2. 有序列表

### 基本语法

1. 第一项
2. 第二项
3. 第三项

说明：数字加点和空格创建有序列表，序号会自动递增

### 嵌套有序列表

1. 第一项
   1. 子项1
   2. 子项2
2. 第二项

说明：有序列表也可嵌套，使用三个空格缩进

## 3. 任务列表

### 基本语法

- [x] 已完成任务
- [ ] 未完成任务
- [ ] 另一个任务

说明：[x]表示完成，[ ]表示未完成，GitHub等平台支持渲染为复选框

## 4. 链接

### 基本链接

[链接文字](URL地址)

说明：方括号内是显示文字，圆括号内是URL地址

### 带标题提示

[百度](https://www.baidu.com "百度搜索")

说明：圆括号内URL后可加空格和引号包裹的标题提示

### 参考式链接

[链接文字][标识符]

[标识符]: URL地址 "标题"

说明：参考式链接将URL定义在文档底部，便于管理多个链接

## 5. 图片

### 基本语法

![替代文字](图片URL)

说明：与链接语法类似，前面加感叹号表示图片

### 带标题

![Logo](https://example.com/logo.png "网站Logo")

说明：图片也可添加标题提示，鼠标悬停时显示
