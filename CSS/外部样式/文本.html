CSS外部样式

1. 引入方式

说明：通过<link>标签引入外部CSS文件

语法：<link rel="stylesheet" href="文件路径">

2. 实际效果展示

这是普通段落（灰色14px）

这是高亮段落（黄色背景）

3. 多文件引入

说明：可引入多个CSS文件，按顺序加载

<link rel="stylesheet" href="reset.css">

<link rel="stylesheet" href="main.css">

4. @import方式

说明：在<style>标签或CSS文件中使用@import导入

@import url("style.css");

5. 特点与适用场景

优点：样式完全分离，可复用，便于缓存

缺点：需要额外HTTP请求

适用：多页面项目、大型网站
