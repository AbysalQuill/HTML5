JavaScript 正则表达式

1. 创建正则表达式

字面量语法

let pattern = /abc/; // 匹配字符串中的abc
let caseInsensitive = /hello/i; // i表示忽略大小写

说明：使用斜杠//创建正则表达式，可以设置修饰符

构造函数语法

let pattern = new RegExp("abc");
let caseInsensitive = new RegExp("hello", "i");

说明：使用RegExp构造函数动态创建正则表达式

2. 常用方法

test()方法

let pattern = /\\d+/;
let result = pattern.test("123"); // true
let result2 = pattern.test("abc"); // false

说明：测试字符串是否匹配正则表达式，返回布尔值

exec()方法

let pattern = /\\w+/g;
let match = pattern.exec("Hello World");
console.log(match[0]); // "Hello"

说明：执行匹配并返回匹配结果数组

match()和replace()方法

let text = "Hello 2024 World";
let numbers = text.match(/\\d+/g); // ["2024"]
let replaced = text.replace(/\\d+/, "2025"); // "Hello 2025 World"

说明：字符串的match和replace方法与正则表达式配合使用

3. 元字符和特殊字符

字符类

/\\d/ // 匹配数字 [0-9]
/\\w/ // 匹配字母、数字、下划线 [a-zA-Z0-9_]
/\\s/ // 匹配空白字符
/[abc]/ // 匹配a、b或c
/[^abc]/ // 匹配不是a、b、c的字符

说明：常用的字符类简写形式

量词

/\\d+/ // 匹配一个或多个数字
/\\d*/ // 匹配零个或多个数字
/\\d?/ // 匹配零个或一个数字
/\\d{3}/ // 匹配恰好3个数字
/\\d{2,5}/ // 匹配2到5个数字

说明：控制匹配字符数量的量词

锚点

/^start/ // 匹配以start开头的字符串
/end$/ // 匹配以end结尾的字符串
/\\bword\\b/ // 匹配完整的单词word

说明：控制匹配位置的锚点

4. 实际应用示例

邮箱验证

function validateEmail(email) {
    let pattern = /^[^\\s@]+@[^\\s@]+\\.[^\\s@]+$/;
    return pattern.test(email);
}

说明：基本的邮箱格式验证

电话号码验证

function validatePhone(phone) {
    let pattern = /^1[3-9]\\d{9}$/;
    return pattern.test(phone);
}

说明：中国手机号码格式验证

密码强度检查

function checkPassword(password) {
    let hasLetter = /[a-zA-Z]/.test(password);
    let hasNumber = /\\d/.test(password);
    let hasSpecial = /[!@#$%^&*()]/.test(password);
    return hasLetter && hasNumber && hasSpecial;
}

说明：检查密码是否包含字母、数字和特殊字符
