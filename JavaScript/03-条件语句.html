JavaScript 条件语句

1. if 语句

基本 if 语句

if (condition) {
    // 条件为真时执行的代码
}

说明：当条件为true时执行代码块

if...else 语句

if (condition) {
    // 条件为真时执行的代码
} else {
    // 条件为假时执行的代码
}

说明：根据条件真假执行不同的代码块

if...else if...else 语句

if (condition1) {
    // 条件1为真时执行
} else if (condition2) {
    // 条件2为真时执行
} else {
    // 所有条件都为假时执行
}

说明：用于多个条件的判断

2. switch 语句

基本 switch 语句

switch (expression) {
    case value1:
        // 代码
        break;
    case value2:
        // 代码
        break;
    default:
        // 默认代码
}

说明：根据表达式的值执行对应的case代码块

3. 三元运算符

条件 ? 值1 : 值2

let result = (age >= 18) ? "成年人" : "未成年人";

说明：简洁的条件表达式，条件为真返回值1，否则返回值2

4. 实际应用示例

成绩等级判断

let score = 85;
let grade;
if (score >= 90) {
    grade = "A";
} else if (score >= 80) {
    grade = "B";
} else if (score >= 70) {
    grade = "C";
} else if (score >= 60) {
    grade = "D";
} else {
    grade = "F";
}

说明：根据分数判断等级

星期判断

let day = 3;
let dayName;
switch (day) {
    case 1: dayName = "星期一"; break;
    case 2: dayName = "星期二"; break;
    case 3: dayName = "星期三"; break;
    case 4: dayName = "星期四"; break;
    case 5: dayName = "星期五"; break;
    case 6: dayName = "星期六"; break;
    case 7: dayName = "星期日"; break;
    default: dayName = "无效的星期";
}

说明：根据数字判断星期几
