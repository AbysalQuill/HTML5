JavaScript 本地存储

1. localStorage

存储数据

localStorage.setItem("username", "张三");
localStorage.setItem("theme", "dark");

说明：在浏览器中永久保存数据，除非用户清除缓存

读取数据

let username = localStorage.getItem("username");
console.log(username); // "张三"

说明：从localStorage获取存储的数据

删除数据

localStorage.removeItem("username"); // 删除单个项目
localStorage.clear(); // 清空所有数据

说明：删除指定的存储项或清空所有数据

2. sessionStorage

会话级存储

sessionStorage.setItem("tempData", "临时数据");
let data = sessionStorage.getItem("tempData");

说明：数据只在当前会话期间有效，关闭页面后自动清除

3. JSON序列化

存储对象和数组

let user = {
    name: "张三",
    age: 25,
    hobbies: ["阅读", "运动"]
};

// 存储前需要转换为JSON字符串
localStorage.setItem("user", JSON.stringify(user));

// 读取后需要解析回对象
let storedUser = JSON.parse(localStorage.getItem("user"));
console.log(storedUser.name); // "张三"

说明：复杂数据类型需要先序列化为JSON字符串再存储

4. 实际应用示例

记住用户偏好

function saveTheme(theme) {
    localStorage.setItem("theme", theme);
}

function getTheme() {
    return localStorage.getItem("theme") || "light";
}

说明：保存用户的主题偏好设置

购物车功能

function addToCart(product) {
    let cart = JSON.parse(localStorage.getItem("cart")) || [];
    cart.push(product);
    localStorage.setItem("cart", JSON.stringify(cart));
}

说明：将购物车商品保存在本地存储中

表单数据暂存

function saveFormData(formData) {
    sessionStorage.setItem("formDraft", JSON.stringify(formData));
}

说明：在用户填写表单时暂存数据，防止丢失
