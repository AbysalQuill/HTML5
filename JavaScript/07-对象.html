JavaScript 对象

1. 创建对象

字面量创建

let person = {
    name: "张三",
    age: 25,
    city: "北京"
};

说明：使用花括号{}创建对象，键值对形式

构造函数创建

function Person(name, age) {
    this.name = name;
    this.age = age;
}
let person = new Person("李四", 30);

说明：使用构造函数创建对象实例

ES6类语法

class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }
}
let person = new Person("王五", 28);

说明：ES6引入的面向对象语法

2. 访问属性

点表示法

console.log(person.name);
console.log(person.age);

说明：使用点号.访问对象的属性

方括号表示法

console.log(person["name"]);
console.log(person["age"]);

说明：使用方括号[]访问对象的属性

3. 添加和删除属性

添加属性

person.city = "上海";
person["email"] = "zhang@example.com";

说明：动态添加新属性到对象中

删除属性

delete person.city;
delete person["email"];

说明：使用delete关键字删除对象属性

4. 对象方法

在对象内定义方法

let calculator = {
    add: function(a, b) {
        return a + b;
    },
    multiply: function(a, b) {
        return a * b;
    }
};

说明：在对象内部定义函数作为方法

箭头函数方法（注意this绑定）

let car = {
    brand: "Toyota",
    start: () => {
        console.log(this.brand + "发动了");
    }
};

说明：箭头函数的this指向定义时的上下文

5. 遍历对象

for...in 循环

for (let key in person) {
    console.log(key + ": " + person[key]);
}

说明：遍历对象的属性名

Object.keys() / Object.values()

let keys = Object.keys(person);
let values = Object.values(person);
console.log("属性:", keys);
console.log("值:", values);

说明：获取对象的所有键或值
