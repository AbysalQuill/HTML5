JavaScript 函数

1. 函数声明

基本函数声明

function greet(name) {
    return "你好, " + name + "!";
}

说明：使用function关键字声明函数，可以返回值

函数表达式

let greet = function(name) {
    return "你好, " + name + "!";
};

说明：将函数赋值给变量，匿名函数形式

箭头函数

let greet = (name) => "你好, " + name + "!";

说明：ES6引入的简洁语法，更短的函数写法

2. 函数参数

必需参数

function add(a, b) {
    return a + b;
}

说明：函数定义时的形参

默认参数

function greet(name = "朋友") {
    return "你好, " + name + "!";
}

说明：当没有传入参数时使用默认值

剩余参数

function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
}

说明：收集所有参数到数组中

3. 作用域

局部变量

function example() {
    let localVar = "局部变量";
    console.log(localVar); // 可以访问
}
console.log(localVar); // 报错: 未定义

说明：函数内部定义的变量只能在函数内访问

全局变量

let globalVar = "全局变量";
function example() {
    console.log(globalVar); // 可以访问
}

说明：在函数外部定义的变量可以在函数内访问

4. 实际应用示例

数学运算函数

function calculator(operation, a, b) {
    switch (operation) {
        case 'add': return a + b;
        case 'subtract': return a - b;
        case 'multiply': return a * b;
        case 'divide': return a / b;
        default: return NaN;
    }
}

说明：根据操作类型执行不同的计算

验证函数

function validateEmail(email) {
    const pattern = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
    return pattern.test(email);
}

说明：使用正则表达式验证邮箱格式
