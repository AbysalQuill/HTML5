JavaScript 数组

1. 创建数组

字面量创建

let fruits = ["苹果", "香蕉", "橙子"];
let numbers = [1, 2, 3, 4, 5];
let mixed = ["文本", 123, true, null];

说明：使用方括号[]创建数组，元素可以是不同类型

构造函数创建

let fruits = new Array("苹果", "香蕉", "橙子");
let numbers = new Array(1, 2, 3, 4, 5);

说明：使用Array构造函数创建数组

2. 数组操作

访问元素

let fruits = ["苹果", "香蕉", "橙子"];
console.log(fruits[0]); // "苹果"
console.log(fruits.length); // 3

说明：通过索引访问数组元素，索引从0开始

修改元素

fruits[1] = "葡萄";
console.log(fruits); // ["苹果", "葡萄", "橙子"]

说明：通过索引修改数组元素的值

添加元素

fruits.push("西瓜"); // 末尾添加
fruits.unshift("芒果"); // 开头添加
console.log(fruits); // ["芒果", "苹果", "葡萄", "橙子", "西瓜"]

说明：push在末尾添加，unshift在开头添加

删除元素

fruits.pop(); // 删除末尾元素
fruits.shift(); // 删除开头元素
fruits.splice(1, 1); // 从索引1开始删除1个元素

说明：pop删除末尾，shift删除开头，splice删除指定位置

3. 数组方法

遍历数组

// forEach - 遍历每个元素
fruits.forEach(function(item) {
    console.log(item);
});

// map - 映射新数组
let upperFruits = fruits.map(function(item) {
    return item.toUpperCase();
});

说明：forEach遍历，map创建新数组

查找元素

// find - 找到第一个符合条件的元素
let firstApple = fruits.find(function(item) {
    return item === "苹果";
});

// filter - 过滤符合条件的元素
let shortFruits = fruits.filter(function(item) {
    return item.length <= 2;
});

说明：find返回第一个匹配项，filter返回所有匹配项

其他常用方法

// slice - 截取数组
let subArray = fruits.slice(1, 3);

// concat - 合并数组
let combined = fruits.concat(numbers);

// sort - 排序
let sortedNumbers = numbers.sort((a, b) => a - b);

说明：slice截取部分，concat合并数组，sort排序
