# JavaScript 学习资料

本项目包含完整的 JavaScript 学习教程，按照知识点分类组织。

## 目录结构

- `01-变量与数据类型.html` - 基础数据类型和变量声明
- `02-运算符.html` - 算术、比较、逻辑运算符
- `03-条件语句.html` - if、switch 语句和三元运算符
- `04-循环语句.html` - for、while 循环和循环控制
- `05-函数.html` - 函数声明、参数和作用域
- `06-数组.html` - 数组操作和常用方法
- `07-对象.html` - 对象创建、属性和方法
- `08-事件处理.html` - 鼠标、键盘和表单事件
- `09-DOM操作.html` - 选择、修改和操作DOM元素
- `10-异步编程.html` - Promise、async/await 和并发请求
- `11-本地存储.html` - localStorage 和 sessionStorage 使用
- `12-错误处理.html` - try-catch 和自定义错误处理
- `13-正则表达式.html` - 字符串匹配和验证
- `14-现代JavaScript.html` - ES6+ 新特性
- `15-模块化开发.html` - ES6 模块系统

## 使用方法

1. 打开任意 HTML 文件即可查看对应教程
2. 在浏览器中按 F12 打开开发者工具查看控制台输出
3. 所有示例代码都包含实际运行效果

## 技术特点

- 遵循 HTML 文件夹的命名规范（数字前缀 + 中文标题）
- 每个文件都是独立的完整示例
- 包含详细的说明文字和实际应用案例
- 支持现代浏览器直接运行
