JavaScript 运算符

1. 算术运算符

加法 +

let sum = 5 + 3; // 8

说明：用于数字相加或字符串连接

减法 -

let diff = 10 - 4; // 6

说明：用于数字相减

乘法 *

let product = 6 * 7; // 42

说明：用于数字相乘

除法 /

let quotient = 20 / 4; // 5

说明：用于数字相除

取余 %

let remainder = 10 % 3; // 1

说明：返回除法后的余数

递增 ++

let x = 5;
x++; // 6

说明：变量值增加1

递减 --

let y = 10;
y--; // 9

说明：变量值减少1

2. 赋值运算符

简单赋值 =

let x = 10;

说明：将右值赋给左变量

复合赋值 +=, -=, *=, /=

x += 5; // 相当于 x = x + 5

说明：运算后赋值，简写形式

3. 比较运算符

相等 ==

5 == "5"; // true

说明：比较值是否相等，忽略类型转换

严格相等 ===

5 === "5"; // false

说明：比较值和类型是否都相等

不相等 !=

5 != "5"; // false

说明：比较值是否不相等

严格不相等 !==

5 !== "5"; // true

说明：比较值或类型是否不相等

大于 >, 小于 <

10 > 5; // true
5 < 10; // true

说明：比较数值大小

大于等于 >=, 小于等于 <=

10 >= 10; // true
5 <= 4; // false

说明：比较数值大小，包含等于情况

4. 逻辑运算符

与 &&

true && true; // true
true && false; // false

说明：两个条件都为真时返回真

或 ||

true || false; // true
false || false; // false

说明：至少一个条件为真时返回真

非 !

!true; // false
!false; // true

说明：反转布尔值

5. 字符串运算符

字符串连接 +

let message = "Hello" + " " + "World"; // "Hello World"

说明：使用+号连接字符串

模板字符串

let name = "张三";
let greeting = `你好, ${name}!`; // "你好, 张三!"

说明：使用反引号和${}插入变量
