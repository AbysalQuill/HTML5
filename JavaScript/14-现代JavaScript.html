JavaScript 现代特性 (ES6+)

1. let和const

块级作用域

if (true) {
    let x = 10; // 只在if块内有效
    const y = 20; // 常量，块级作用域
}
console.log(x); // ReferenceError: x is not defined

说明：let和const具有块级作用域，不同于var的函数作用域

const的限制

const person = {name: "张三"};
person.name = "李四"; // 可以修改对象属性
// person = {name: "王五"}; // 不能重新赋值整个对象

说明：const保证变量引用不变，但对象内容可以修改

2. 模板字符串

多行字符串

let message = `这是
多行
字符串`;

说明：使用反引号``创建多行字符串

表达式插值

let name = "张三";
let age = 25;
let greeting = `你好, ${name}!
你今年${age}岁了。`;

说明：使用${}在字符串中插入变量或表达式

3. 箭头函数

简洁语法

// 传统函数
function add(a, b) {
    return a + b;
}

// 箭头函数
let add = (a, b) => a + b;

说明：箭头函数提供更简洁的语法

this绑定

class Timer {
    constructor() {
        this.seconds = 0;
    }
    
    start() {
        setInterval(() => {
            this.seconds++; // 正确访问this
            console.log(this.seconds);
        }, 1000);
    }
}

说明：箭头函数继承外层this，避免this指向问题

4. 解构赋值

数组解构

let [first, second] = ["苹果", "香蕉"];
console.log(first); // "苹果"
console.log(second); // "香蕉"

let [x, , z] = [1, 2, 3, 4]; // 跳过第二个元素

说明：从数组中提取值并赋值给变量

对象解构

let person = {name: "张三", age: 25};
let {name, age} = person;
console.log(name); // "张三"
console.log(age); // 25

// 重命名
let {name: fullName, age: years} = person;

说明：从对象中提取属性并赋值给变量

5. 展开运算符

数组展开

let arr1 = [1, 2, 3];
let arr2 = [...arr1, 4, 5]; // [1, 2, 3, 4, 5]

let max = Math.max(...arr1); // 展开数组作为参数

说明：...操作符展开数组或对象

对象展开

let person = {name: "张三"};
let updatedPerson = {...person, age: 25, city: "北京"};

说明：合并对象属性

6. 类语法

基本类定义

class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }
    
    greet() {
        return `你好, 我是${this.name}`;
    }
    
    static createDefault() {
        return new Person("无名", 0);
    }
}

说明：ES6引入的面向对象编程语法

继承

class Student extends Person {
    constructor(name, age, grade) {
        super(name, age);
        this.grade = grade;
    }
    
    study() {
        return `${this.name}正在学习`;
    }
}

说明：使用extends实现类的继承
