JavaScript 模块化开发

1. 模块基础概念

什么是模块

// math.js - 数学工具模块
export function add(a, b) {
    return a + b;
}

export function subtract(a, b) {
    return a - b;
}

说明：模块是包含特定功能的独立文件，可以导出和导入功能

模块的封装性

// user.js - 用户管理模块
let userId = 0; // 私有变量

function generateId() {
    return ++userId;
}

export function createUser(name) {
    return {
        id: generateId(),
        name: name,
        createdAt: new Date()
    };
}

说明：模块内部变量和方法默认私有，通过export暴露公共接口

2. ES6 模块语法

导出模块

// utils.js
export const PI = 3.14159;

export function formatDate(date) {
    return date.toISOString().split('T')[0];
}

export default function logger(message) {
    console.log(`[${new Date().toISOString()}] ${message}`);
}

说明：使用export关键字导出变量、函数或默认导出

导入模块

import { PI, formatDate } from './utils.js';
import logger from './utils.js';

// 重命名导入
import { formatDate as format } from './utils.js';

说明：使用import语句导入其他模块的功能

3. 实际应用示例

计算器应用

// calculator.js
export class Calculator {
    constructor() {
        this.history = [];
    }

    add(a, b) {
        const result = a + b;
        this.history.push(`add(${a}, ${b}) = ${result}`);
        return result;
    }

    multiply(a, b) {
        const result = a * b;
        this.history.push(`multiply(${a}, ${b}) = ${result}`);
        return result;
    }

    getHistory() {
        return [...this.history];
    }
}

说明：创建可复用的计算器类模块

配置管理

// config.js
const API_URL = 'https://api.example.com';
const TIMEOUT = 5000;

export const Config = {
    API_URL,
    TIMEOUT,
    getApiUrl(path = '') {
        return `${API_URL}${path}`;
    }
};

说明：集中管理应用的配置信息
