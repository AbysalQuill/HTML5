JavaScript 异步编程

1. 回调函数（Callback）

基本回调模式

function fetchData(callback) {
    setTimeout(() => {
        callback("数据获取成功");
    }, 1000);
}

fetchData(function(result) {
    console.log(result);
});

说明：将函数作为参数传递，在异步操作完成后调用

回调地狱问题

getData(function(data1) {
    processData(data1, function(result1) {
        finalProcess(result1, function(finalResult) {
            console.log(finalResult);
        });
    });
});

说明：多个异步操作嵌套导致的代码难以维护

2. Promise对象

创建Promise

let promise = new Promise((resolve, reject) => {
    setTimeout(() => {
        if (Math.random() > 0.5) {
            resolve("操作成功");
        } else {
            reject("操作失败");
        }
    }, 1000);
});

promise.then(result => {
    console.log(result);
}).catch(error => {
    console.log(error);
});

说明：Promise表示一个可能在未来完成的操作

链式调用

fetchData()
    .then(data => processData(data))
    .then(result => finalProcess(result))
    .then(finalResult => console.log(finalResult))
    .catch(error => console.log("错误:", error));

说明：使用then和catch方法处理异步结果

3. async/await语法

async函数

async function getData() {
    try {
        let result = await fetchData();
        console.log(result);
    } catch (error) {
        console.log("错误:", error);
    }
}

说明：async函数返回Promise，可以使用await等待结果

await表达式

async function processMultiple() {
    let data1 = await fetchData1();
    let data2 = await fetchData2(data1);
    return processFinal(data2);
}

说明：await只能在async函数中使用，暂停执行直到Promise完成

4. 实际应用示例

模拟API请求

function apiCall(url) {
    return new Promise(resolve => {
        setTimeout(() => {
            resolve(`从 ${url} 获取的数据`);
        }, Math.random() * 2000 + 500);
    });
}

说明：模拟网络请求的异步操作

并发请求

async function fetchAllData() {
    let [data1, data2, data3] = await Promise.all([
        apiCall("api/users"),
        apiCall("api/posts"), 
        apiCall("api/comments")
    ]);
    return { users: data1, posts: data2, comments: data3 };
}

说明：使用Promise.all并行执行多个异步操作
