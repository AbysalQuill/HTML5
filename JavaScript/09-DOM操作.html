JavaScript DOM操作

1. 选择元素

通过ID选择

let element = document.getElementById("myId");

说明：通过元素的id属性获取单个元素

通过类名选择

let elements = document.getElementsByClassName("myClass");
// 或
let elements = document.querySelectorAll(".myClass");

说明：通过class属性获取元素集合

通过标签名选择

let divs = document.getElementsByTagName("div");

说明：通过HTML标签名获取元素集合

通过CSS选择器

let element = document.querySelector("#myId");
let elements = document.querySelectorAll("div.myClass");

说明：使用CSS选择器语法选择元素

2. 修改内容

修改文本内容

element.textContent = "新的文本内容";
element.innerHTML = "<strong>加粗文本</strong>";

说明：textContent设置纯文本，innerHTML设置HTML内容

修改属性

element.src = "new-image.jpg";
element.href = "https://example.com";
element.className = "new-class";

说明：直接修改元素的属性值

3. 创建和添加元素

创建新元素

let newDiv = document.createElement("div");
let newText = document.createTextNode("新文本");

说明：使用createElement创建新元素，createTextNode创建文本节点

添加到DOM

parent.appendChild(newDiv);
parent.insertBefore(newDiv, referenceElement);
newDiv.replaceWith(replacementElement);

说明：appendChild添加到最后，insertBefore在指定元素前插入

4. 样式操作

修改CSS样式

element.style.color = "red";
element.style.fontSize = "20px";
element.style.backgroundColor = "#f0f0f0";

说明：直接修改元素的style属性来改变样式

切换类名

element.classList.add("active");
element.classList.remove("inactive");
element.classList.toggle("highlight");

说明：使用classList方法管理元素的类名

5. 实际应用示例

动态列表

function addListItem(text) {
    let ul = document.getElementById("myList");
    let li = document.createElement("li");
    li.textContent = text;
    ul.appendChild(li);
}

说明：动态创建列表项并添加到列表中

表单验证

function validateForm() {
    let input = document.getElementById("username");
    if (input.value.length < 3) {
        input.style.borderColor = "red";
        return false;
    }
    return true;
}

说明：根据输入内容动态修改元素样式
