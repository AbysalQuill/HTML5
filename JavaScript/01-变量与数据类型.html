JavaScript 变量与数据类型

1. 变量声明

let 声明

let username = "张三";

说明：let用于声明可重新赋值的变量

const 声明

const PI = 3.14;

说明：const用于声明常量，不能重新赋值

var 声明（传统方式）

var age = 18;

说明：var是ES5的变量声明方式，作用域不同

2. 数据类型

字符串 (String)

let username = "张三";

说明：用于表示文本数据，使用单引号或双引号

数字 (Number)

let age = 18;
let price = 99.99;

说明：包括整数和小数，JavaScript只有一种数字类型

布尔值 (Boolean)

let isStudent = true;
let isAdult = false;

说明：表示真或假，常用于条件判断

空值 (null)

let score = null;

说明：表示空值或无值，typeof返回"object"

未定义 (undefined)

let address;

说明：声明但未赋值的变量，值自动为undefined

3. 复杂数据类型

数组 (Array)

let hobbies = ["读书", "运动", "音乐"];

说明：用于存储多个值的列表，从0开始索引

对象 (Object)

let person = {name: "李四", age: 20};

说明：用于存储键值对，表示复杂数据结构

4. 变量重新赋值

age = 19; // 正确
// PI = 3.14159; // 错误: const不能重新赋值

说明：let声明的变量可以重新赋值，const声明的常量不能
