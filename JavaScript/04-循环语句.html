JavaScript 循环语句

1. for 循环

基本 for 循环

for (let i = 0; i < 5; i++) {
    console.log(i);
}

说明：初始化; 条件判断; 增量表达式，常用于已知次数的循环

for...in 循环

let person = {name: "张三", age: 20};
for (let key in person) {
    console.log(key + ": " + person[key]);
}

说明：遍历对象的属性名

for...of 循环

let fruits = ["苹果", "香蕉", "橙子"];
for (let fruit of fruits) {
    console.log(fruit);
}

说明：遍历数组或可迭代对象的值

2. while 循环

基本 while 循环

let count = 0;
while (count < 3) {
    console.log(count);
    count++;
}

说明：当条件为真时重复执行代码块

do...while 循环

let num = 0;
do {
    console.log(num);
    num++;
} while (num < 3);

说明：先执行一次代码，然后检查条件

3. 循环控制语句

break 语句

for (let i = 0; i < 10; i++) {
    if (i === 5) {
        break; // 跳出整个循环
    }
    console.log(i);
}

说明：立即终止循环的执行

continue 语句

for (let i = 0; i < 5; i++) {
    if (i === 2) {
        continue; // 跳过本次循环
    }
    console.log(i);
}

说明：跳过当前循环的剩余代码，进入下一次循环

4. 实际应用示例

求和计算

let sum = 0;
for (let i = 1; i <= 10; i++) {
    sum += i;
}
console.log("1到10的和:", sum);

说明：计算1到10的和

查找最大值

let numbers = [3, 7, 2, 9, 1];
let max = numbers[0];
for (let i = 1; i < numbers.length; i++) {
    if (numbers[i] > max) {
        max = numbers[i];
    }
}
console.log("最大值:", max);

说明：在数组中查找最大值
