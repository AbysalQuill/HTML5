HTML5表单进阶

1. form标签核心属性

action属性（提交地址）

说明：action指定表单数据提交的URL地址

method属性（GET方式）

说明：GET将参数拼接在URL后，适合查询操作，有长度限制

method属性（POST方式）

说明：POST将参数放在请求体中，适合提交敏感或大量数据

2. 单选框 radio

选择性别：

说明：同name的单选框只能选一个，实现互斥选择

3. 复选框 checkbox

选择兴趣爱好：

说明：复选框可多选，checked属性设置默认选中

4. 隐藏域 hidden

说明：hidden字段用户不可见，但会随表单提交，常用于传递ID等隐藏数据

5. 文件上传 file

说明：type="file"允许选择文件，enctype必须设为multipart/form-data，accept限制文件类型

6. 日期时间颜色选择器

日期选择 date

说明：弹出日期选择器，格式为YYYY-MM-DD

时间选择 time

说明：弹出时间选择器，格式为HH:MM

日期时间 datetime-local

说明：同时选择日期和时间

颜色选择 color

说明：弹出颜色选择器，返回十六进制颜色值
