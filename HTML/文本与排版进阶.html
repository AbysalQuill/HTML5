HTML5文本与排版进阶

1. 强调文本 strong / em

strong（重要强调）

这是非常重要的内容，浏览器默认加粗显示。

说明：strong表示内容的重要性，语义上比em更强

em（着重强调）

这是需要强调的内容，浏览器默认斜体显示。

说明：em表示内容的着重语气，用于强调语调

2. 删除与插入 del / ins

del（删除文本）

原价：999元

说明：del表示被删除的内容，浏览器默认添加删除线

ins（插入文本）

现价：599元

说明：ins表示新插入的内容，浏览器默认添加下划线

组合使用

价格调整：999元 599元

说明：常用于显示修改前后的对比

3. 上下标 sub / sup

sub（下标）

水的化学式：H2O

说明：sub使文字变小并降低，用于化学式、脚注等

sup（上标）

面积单位：m2，体积单位：m3

说明：sup使文字变小并升高，用于数学幂次、商标等

4. 预格式化文本 pre

    function hello() {
        console.log("Hello");
        console.log("World");
    }

说明：pre保留空格和换行，常用于显示代码或诗歌等需要保持格式的内容

5. 引用标签 blockquote / q

blockquote（块级引用）

学而时习之，不亦说乎？有朋自远方来，不亦乐乎？

——《论语》

说明：blockquote用于长引用，独占一行，通常有缩进效果

q（行内引用）

孔子说：三人行，必有我师焉，这句话广为流传。

说明：q用于短引用，嵌入在段落中，浏览器自动添加引号

6. 其他文本标签

mark（高亮文本）

搜索结果中的关键词会被高亮显示。

说明：mark标记需要突出显示的文本，默认黄色背景

code（代码片段）

使用<div>标签创建容器。

说明：code表示计算机代码，默认等宽字体

small（小号文本）

主要内容（这是补充说明）

说明：small使文本变小，用于免责声明、版权等次要信息
